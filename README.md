# brokenkeys
Keymap for OLKB Preonic Rev 3 with different underglow colours for different layers.

## Layer List
```bash
Layer 0 - QWERTY
Layer 1 - Colemak MOD-DH
Layer 2 - Arrow Keys
Layer 3 - Symbols
Layer 4 - Shortcuts
Layer 5 - Functions
Layer 6 - Settings
```
